### This project created for the educational goal to study Reactive programming with WebFlux

### Used technology
 - WebFlux
 - RSocket as a data transfer protocol between UI and back-end 
 - Docker
 - Reactive Kafka, KafkaStream
 - Avro
 - SpringBoot
 - MongoDB
 - PostgreSQL
